Course Offerings

Curious to find out more? No sure about which course program might be right for you based on your needs? This is your chance...follow the above link and sign up ( it's free so you have nothing to lose! )

Intro to Front End Web Development

This course assumes you have little or no experience in developing web applications. The classes will be held 3 nights a week, 3 hours / night ( class and lab time ) student will have a good understanding of the style languages of HTML and CSS as well as basic logic with JavaScript, and we will provide intros to some client side frameworks and libraries.

Ruby on Rails and Postgresql

The students will spend 2 nights a week for 90 minutes / night. Each student will have a good understanding of the Ruby language, the Rails framework, and Relational Databases with PostgreSQL through Rails Active Record. **This course furthers the skills taught in the front end program and guides the learning through the full development life cycle**
